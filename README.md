# Saloni & Shrenik Engagement Invite - Hybrid Background Version

Run:
```bash
npm install
npm run dev
```

Edit text, links, phone numbers, and image paths in `src/inviteConfig.js`.

Replace artwork files in `public/`:
- `logo.png`
- `couple.png`
- `page1-bg.png`, `page2-bg.png`, `page3-bg.png`
- `map.png`

This version uses full-page background artwork images, then overlays editable HTML text/buttons on top.
